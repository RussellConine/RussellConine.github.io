Projects

I built an e-book reader coded in Python using a Raspberry Pi. The device contains 6 pushbuttons to control the playback of audiobooks stored on a thumbdrive.

The user can start and pause the player; fast forward and rewind; and change between books. The program uses the speech syntheizer eSpeack to read the name and number (based on alphabetical order of the books stored) of the book aloud when it is chosen.

The reader stores the location of the user's progress in each book to a meta file.

I made the device so that someone with very limited sight could listen to audiobooks.
